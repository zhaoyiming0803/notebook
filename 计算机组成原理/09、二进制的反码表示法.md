在计算补码的过程中还是使用了减法

### 引进反码的目的

找出原码和反码之间的规律，消除转换过程中的减法

``` javascript
var x = Number
if (x >=0 && x < Math.pow(2, n)>) {
  x = x
} else if (x >= -Math.pow(2, n) && x < 0>) {
  x = Math.pow(2, n+1) + x - 1
}
```

x = -7, n = 4

原码：x = 10111

反码：x = 2^5 - 7 - 1 = 11000

### 计算规则

原码与补码的首位1表示负数

整数的原码 = 补码 = 反码

负数的反码 = 原码除符号位外按位取反

负数的补码 = 反码 + 1
